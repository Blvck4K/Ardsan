Ardsan is a cooperative management software platform designed to help cooperative societies in Nigeria automate and digitize their administrative and financial operations. 

TECHNOLOGIES USED TO MAKE THIS PROJECT LIVE
-SUPABASE: Used for the database where users datas are stored and managed.
-VERCEL: Used for the hosting of the web application.
-NAME CHEAP: Were the domain was bought for a cheaper and more secured rate.
-ANTIGRAVITY: IDE used with its AI assistant.
